# CampusHub

## 1. 项目简介 (Project Introduction)

- **项目名称**：校园互助服务平台 (CampusHub)
- **课程背景**：软件工程与计算 II
- **核心目标**：旨在解决校园内信息碎片化、匹配效率低的问题，提供统一的互助平台（如快递代取、学习辅导、二手交易等）
