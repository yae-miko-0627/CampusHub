## 2. AI工具评估分析

# AI 编程助手
1. GithubCopilot:
    Github Copilot与各种IDE的集成度高，并拥有学生认证的免费使用权限，具有更长时间的使用经验，与个人编程习惯适配度高，功能覆盖面广，并能与Github平台无缝集成

# AI 对话工具
1. Gemini:
    Gemini侧重于深度分析，复杂信息整合，适用与软件工程项目中的需求分析、架构设计等阶段，能够提供更专业的技术建议和方案，团队成员已有使用经验，能够快速上手。
2. Deepseek:
    Deepseek在国内具有较高的知名度和用户基础，使用门槛较低，适合日常生活中的简单对话和信息查询，能够满足团队成员在项目开发过程中对于快速获取信息和解决问题的需求。
3. ChatGPT:
    ChatGPT在逻辑推理，精准执行与结构化的创意内容生成方面表现出色
